# Lead Tracker Chrome Extension

A simple Chrome extension to save, manage, and access useful links (leads) directly from the browser. Designed for quick bookmarking and retrieval without relying on external tools.

---

## 🚀 Features

- Save current tab URL with one click  
- Manually add custom links  
- Persistent storage using browser localStorage  
- Delete all saved leads  
- Lightweight and fast browser extension  

---

## 🛠 Tech Stack

- JavaScript
- HTML / CSS
- Chrome Extension APIs
- LocalStorage

---

## 📂 Project Structure

- `manifest.json` – Extension configuration  
- `index.html` – UI layout  
- `index.js` – Core logic  
- `index.css` – Styling  

---

## ⚙️ How to Run Locally

1. Download or clone this repository  
2. Open Chrome and go to: `chrome://extensions/`  
3. Enable **Developer Mode**  
4. Click **Load unpacked**  
5. Select the project folder  
6. The extension will appear in your browser  

---

## 🎯 Use Case

Useful for saving frequently visited links, job postings, or resources without switching tabs or using external bookmarking tools.
